# 프로젝트 소개



이 프로젝트는 포트폴리오 목적으로 제작된 도서, 티켓, 각종 상품을 판매하는 가상 쇼핑몰입니다. 

실제 서비스를 염두에 두기보다는, 다양한 기능을 구현하고 풀스택 개발 역량을 보여주기 위해 개발되었습니다.

# 팀원 소개

+ 팀장(황문성)
+ PM(원필재/이이섭)
+ __PL__ (윤해연 / __박근태__)
+ DBA(전희원)
+ 서기(지유빈)
